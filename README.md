# Loyalty Program App
Code challenge from Aerolab

## Dependencies
* React
* Axios
* Sass

## To run locally
* Run `yarn install`
* And then `yarn start`
* Be sure to have your .env file loaded properly
  * It only needs to have the key `REACT_APP_API_URL` set to the proper api url
